# SaaS Customer Churn & Risk Analysis

## Project Overview
Developed an end-to-end Business Intelligence solution to identify at-risk customers.

## Key Features
* **Predictive Modelling**: Built and optimized Machine Learning models (Random Forest, XGBoost).
* **Risk Segmentation**: Categorized over 240 users using DAX measures.
* **Interactive Dashboards**: Created dynamic visualizations in Power BI.

## Tech Stack
* **Language**: Python
* **Libraries**: Scikit-learn, Pandas, XGBoost
* **Visualization**: Power BI
